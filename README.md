# WSPR (Web Secure P2P Relay)

A secure, end-to-end encrypted peer-to-peer communication platform built with privacy and security at its core.

## Overview

WSPR is a modern web application that provides secure messaging, video calling, file sharing, and group communication capabilities. Built with state-of-the-art encryption and security practices, WSPR ensures your communications remain private and secure.

## Key Features

- End-to-end encrypted messaging
- Secure video calls and conferences
- File sharing with encryption
- Group messaging and management
- Contact management
- User profiles
- Screen sharing capabilities

## Security Features

- End-to-end encryption using Signal Protocol
- Perfect forward secrecy
- Zero-knowledge architecture
- Secure WebRTC implementation
- Encrypted data storage
- Protection against MITM attacks
